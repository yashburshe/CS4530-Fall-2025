### Learning Objectives:
In this module, you will learn to:
* describe the basic ideas of the following architectures, with examples and pictures:
  * anarchic
  * layered
  * pipeline
  * event-driven
  * microkernel
  * microservice
* describe the main features of the following communication modalities:
  * procedure calls
  * HTTP and REST
  * Websockets
